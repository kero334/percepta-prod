# Percepta Website Code

Thank you for downloading the Percepta website source code!

## Installation

1. Install dependencies:
   ```bash
   npm install
   # or
   pnpm install
   ```

2. Set up environment variables:
   - Copy `.env.example` to `.env`
   - Add your Convex deployment URL

3. Run development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## Tech Stack

- React 18 + TypeScript
- Vite
- Convex (Backend/Database)
- Tailwind CSS
- shadcn/ui components
- Framer Motion
